# brary handbook

## script shorthands:
### space shorthands:
- `X=` - x location on the board
- `Y=` - y location on the board
- `T=` - space texture
- `C=` - space color
### board shorthands:
- `W=` - board max x position (width)
- `H=` - board max y position (height)
- `T=` - background texture
- `C=` - background color
### player shorthands:
- `S=` - player speed
